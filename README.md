## Features

- Encrypt plain text using a secret key  
- Decrypt encrypted text using the same key  
- Fully client-side (no backend required)  
- Works in all modern web browsers  
- Simple and clean user interface

## How to Use

### Encryption
1. Enter the text you want to encrypt.
2. Enter a secret key.
3. Click the **Encrypt** button.
4. Copy the encrypted output.

### Decryption
1. Paste the encrypted text.
2. Enter the same secret key.
3. Click the **Decrypt** button.
4. The original text will be displayed.

## Notes
- The same key must be used for both encryption and decryption.
- If the secret key is lost, the encrypted data cannot be recovered.
- All encryption and decryption happens inside the browser.
- No data is stored, tracked, or sent to any server.
- This project is intended for learning and basic use, not for high-security or production environments.
